# Sea_battle
Console game - sea battle

### Create Python virtual environment 
```
python3 -m venv venv
```
### Activate it
Run from `${project_root}:
```
. venv/bin/activate
```
### To deactivate run:
```
deactivate
```
### Install dependencies
Run from `${project_root}:
```
pip install -r requirements.txt
```
### How to play
Run from `${project_root}:
```
python run.py
```
